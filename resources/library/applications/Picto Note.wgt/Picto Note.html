Pour importer une image, il vous suffit de faire un drag&drop de l'image depuis votre bibliothèque d'images, sur la fenêtre du widget PictoNote.

Pour annoter l'image faite un simple click où vous désirez voir apparaître une punaise contenant l'annotation.

Si vous désirez déplacer une punaise, il vous suffit de tirer la punaise précédemment crée jusqu'à ça nouvelle position.
