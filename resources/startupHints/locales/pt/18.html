Open-Sankoré não está disponível em modo de janela. O software sempre usará toda a superfície de sua tela.

No entanto, é possível alternar entre o modo “desktop” ou “máscara”.

O modo “desktop” é acessível clicando o ícone “Desktop” na barra de ferramentas.

Para ocultar o Open-Sankoré você precisará usar o comando “Ctrl+H” no Windows e Linux ou “Cmd+H” no MacOS.

Você também poderá usar o comando “Ctrl+Tab” no Windows ou Linux ou “Cmd+Tab” no MacOS para passar de uma aplicação para outra para fazer um copia e cola, por exemplo.
