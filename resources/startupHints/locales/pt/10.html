Pode-se adicionar lligações multimídia a objetos que contém imagens fixas ou texto.

O menu permite adicionar  :

  • uma ligação a uma página web (requer uma conexão de internet),

  • uma ligação para outra página da apresentação,

  • uma ligação quer permite tocar um áudio quando clicar em um objeto.

Para adicionar uma ação, selecione a opção no menu do quadro do objeto. Clique em « OK » para validar sua ação.

Para remover a ação, use novamente o mesmo menu.
