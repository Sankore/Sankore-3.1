Se você fizer um clique longo no botão «Páginas» na barra de ferramentas, um menu abrirá para você escolher se quer adicionar uma nova página, duplicar a página atual ou importar uma página ou um conjunto de páginas de um arquivo em seu computador. Estes arquivos possuem ao menos uma página no formato de imagem (png, jpg, gif, …) ou .pdf.

OBS : para a página de título (page 0), apenas duas opções estão disponíveis (nova página e importar página).
