A ferramenta de interação com objetos (dedo mágico) permite mover objetos sem mostrar seu quadro cinza. Esta interface é melhor usada para o trabalho com os alunos para que não tenham que usar uma interface muito complexa.

Esta ferramenta permite também executar algumas ações (ligações hipertexto, ligações para outras páginas, tocar um som) e interagir com aplicações, utilitários e animações flash.
