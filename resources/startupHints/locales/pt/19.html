Um novo documento é criado cada vez que você executa o Open-Sankoré. Ele aparecerá na pasta «Documentos sem nome» no modo «Documentos» (coluna da esquerda).

Para facilitar o trabalho dos usuários, o Open-Sankoré gravará automaticamente seu trabalho cada vez que uma página é modificada ou o software é fechado. Portanto, não existe um botão com a função “Gravar”.

A pasta “Modelos” permite deixar apresentações que você gostaria de preservar sem modificações. Você precisará arrastá-las para outra pasta para duplicá-las e então editá-las.

No modo “Documento”, é possível :

  • Renomear um documento

  • Criar pastas e sub-pastas

  • Importar ou exportar documentos do Open-Sankoré (*.ubz) ou pastas contendo documentos (*.ubx)

  • Adicionar .pdf em suas páginas

  • Adicionar pastas de imagens

  • Reutilizar páginas de uma apresentação

  • ...
